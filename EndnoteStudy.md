## Reference
  ### DOI
  - DOI(Digital Object Identifier):数字对象识别码，识别数字资源，包括视频、报告、书。
  - 将**资源**统一命名为一个**识别号**，并将**识别号**在解析为**具体地址**。
  - 例子：通过https://doi.org/xx.xxxx/doi号 可直接下载对应资源
  ### 写法规范
   - Author Name
      - 姓氏全写、名字缩写
      - Endnote作者姓名地两种写法
        - 名字+姓氏
        - 姓氏，名字
